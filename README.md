## Who I Am

I am Jacob Johnston, founder of True North Data Strategies LLC (SDVOSB), building operations systems that become production software.

My path is operations-first and self-taught engineering second. The work below shows both sides: process control, compliance rigor, and real implementation across Apps Script, TypeScript, Python/FastAPI, and cloud delivery.
